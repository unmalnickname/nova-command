# Nova Command - Complete Platform Specification

## Vision Statement

Nova Command is a native desktop application for managing AI agent teams. It combines Discord-like communication (channels, chat, voice), Codex-style agent monitoring (real-time task streaming), and Kanban project management into one unified interface.

**Tagline:** "Your AI Team's Command Center"

---

## Target Platform

- **Primary:** macOS native app (Tauri)
- **Secondary:** Web app (PWA), Windows, Linux
- **Mobile:** PWA initially, native iOS/Android later

---

## Core Features

### 1. Channels & Chat (Discord-like)
- Text channels organized by topic (#general, #dev, #research, #support)
- Voice channels for multi-agent calls
- Persistent message history with search
- Markdown support in messages
- File/image attachments
- Thread replies within channels
- @mentions for agents and users

### 2. Agent Monitor (Codex-like)
- Real-time view of what each agent is doing
- Streaming output (thinking, tool calls, progress)
- Progress indicators (percentage, status)
- Quick actions (pause, resume, reassign)

### 3. Kanban Board
- Columns: Inbox, Assigned, In Progress, Review, Done, Archived
- Drag-and-drop task movement
- Task cards with assignees, tags, priority
- Quick task creation
- Filter by agent, status, tag

### 4. Voice Rooms
- Multi-agent voice calls (the killer feature)
- Push-to-talk or always-on
- Visual indicators for who's speaking
- Join/leave easily
- Background listening mode

### 5. Agent Roster
- List of all agents with status
- Quick status toggle (active/idle/blocked)
- Agent detail panel (personality, skills, current task)
- Spawn new agents (Agent Factory)

### 6. Memory & Context
- WORKING.md viewer per agent
- Daily notes timeline
- Session history search
- Cross-agent context sharing

### 7. Notifications & Activity
- Real-time activity feed
- @mention notifications
- Thread subscription alerts
- Daily standup summary
- Desktop notifications (native)

---

## User Roles

| Role | Description |
|------|-------------|
| Commander | Full access, can create/delete agents, admin |
| Operator | Can assign tasks, chat, use voice |
| Observer | Read-only, can view but not interact |

```

## Key User Flows

### 1. Send message to agent
1. Select channel from sidebar
2. Type message with @mention
3. Press Enter or click Send
4. Message appears in chat
5. Agent receives notification on next heartbeat
6. Agent responds in channel

### 2. Create and assign task
1. Click "+ New Task" in Kanban
2. Fill title, description
3. Select assignees from agent list
4. Add tags
5. Click Create
6. Task appears in Inbox
7. Drag to Assigned column (or auto-moves on assignment)

### 3. Join voice room
1. Click voice room in sidebar
2. Confirm microphone permission (first time)
3. Join room, see other participants
4. Speak or use push-to-talk
5. See visual indicator when agents speak
6. Click Leave to exit

### 4. Monitor agent progress
1. Click "Agent Monitor" in sidebar
2. See all active agents with streaming output
3. Click agent card to expand
4. Watch real-time progress
5. Use Pause/Stop if needed

### 5. Review daily standup
1. Click notification or "Daily Standup" view
2. See summary of day's activity
3. Click items to jump to detail
4. Click "Send to Telegram" to share
